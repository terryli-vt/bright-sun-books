# bright-sun-books

A full-stack application of online bookstore.

## Frontend

- Vue 3 + TypeScript
- Vue Router
- Tailwind CSS & Daisy UI

### Frontend Setup

Install dependencies:

```sh
npm install
```

Run the development server:

```sh
npm run dev
```

## Backend

- Node.js
- Express
- Drizzle ORM & NeonDB

### Backend Setup

Install dependencies:

```sh
npm install
```

Run the development server:

```sh
npm run dev
```

View Database:

```sh
npm run studio
```
